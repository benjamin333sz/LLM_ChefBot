# Chef Bot LLM

Ceci est notre rendu de TP ChefBot LLM en python. Vous trouverez l'énoncé du TP dans ENONCE_TP.md

Veuillez avant toute chose, créer un environnement virtuel, installer les dépencances du fichier _"requirements.txt"_ et rajouter un fichier _".env"_ pour l'utilisation des clés API langfuse et groq 

Pour éxécuter les codes et voir nos résultats, lancer le fichier _"main.py"_ pour chaque sous-dossier (exception pour la partie 1 où il faut lancer _"chefbot.py"_)
